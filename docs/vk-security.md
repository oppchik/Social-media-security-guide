## Как снизить риск фишинга во ВКонтакте

*   <span class="badge">Пароли</span> **Никогда** не вводите пароль после перехода по ссылкам из ЛС.
*   <span class="badge">Коды</span> Не отправляйте коды подтверждения даже тем, кто представляется вашим знакомым.
*   <span class="badge">Приложение</span> Игнорируйте сообщения о «проверке страницы», если они пришли не через официальные каналы.
*   <span class="badge">Браузер</span> Всегда проверяйте адресную строку: там должно быть строго `vk.com`.
