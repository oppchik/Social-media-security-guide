# О проекте

Этот сайт создан как учебный ресурс по личной кибербезопасности в социальных сетях и мессенджерах. Он помогает разобраться, какие настройки действительно влияют на защиту аккаунта, какие риски встречаются чаще всего и какие повседневные действия снижают вероятность взлома или утечки данных.

## Принципы содержания

- Материалы написаны простым и понятным языком.
- Основной акцент сделан на профилактике, безопасном поведении и цифровой гигиене.
- Рекомендации подходят для разных платформ, но учитывают особенности популярных сервисов.
- Структура сайта позволяет использовать его как краткий справочник, памятку и учебный материал.

## Для кого этот ресурс

- для пользователей, которые хотят лучше защитить свои аккаунты;
- для студентов и школьников, изучающих основы цифровой грамотности;
- для преподавателей, которым нужен наглядный учебный материал;
- для всех, кто хочет быстро повторить базовые правила безопасности.
