# Инструкция по использованию ресурса

Этот сайт устроен как учебный навигатор. Его можно читать последовательно или использовать точечно, когда нужно быстро проверить конкретную платформу.

## Как пользоваться навигацией

- верхнее меню помогает быстро перейти к главным разделам;
- боковое меню показывает полную структуру учебного сайта;
- встроенный поиск VitePress позволяет находить темы по ключевым словам;
- нижняя навигация на странице помогает листать материал по порядку.

## Что можно обновлять в проекте

- тексты страниц и примеры формулировок;
- блоки-заглушки под скриншоты;
- состав чек-листа и его формулировки;
- цветовую схему и карточки в пользовательских стилях;
- навигацию и структуру разделов.

## Ограничение ресурса

Сайт предназначен только для обучения безопасному поведению и профилактике рисков. При обновлении контента сохраняйте этот принцип и не добавляйте материалы, связанные с вредоносной активностью.
